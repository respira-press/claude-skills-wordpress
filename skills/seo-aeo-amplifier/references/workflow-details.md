# SEO & AEO Amplifier - Detailed Workflow

## Detailed Phase Breakdown

### Phase 1: Audit
- Crawl pages/posts
- Detect plugin context (Yoast/Rank Math/none)
- Score on-page issues and AEO opportunities

### Phase 2: Decision
- Present issue counts and potential impact
- Ask for explicit duplicate-creation approval

### Phase 3: Duplicate Fixes
- Create duplicate drafts
- Apply metadata, heading, schema, and alt-text improvements
- Build before/after summaries

### Phase 4: Review
- Review duplicate set in WordPress admin
- Publish selected pages only

## Example Output

Includes health score, severity buckets, top opportunities, and duplicate-first recommendation steps.
